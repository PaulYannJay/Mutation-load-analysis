#This is a collection of scripts and codes for the paper "Mutation load at a mimicry supergene sheds new light on the evolution of inversion polymorphisms"

Please contact me (paul.yann.jay@gmail.com) if you like further detail on what the scripts do or how to use them.


The script to compute Sliding window PCA (Fig 1B) is SWPCA.sh.
Detail on Gap analysis and Transposable element analysis and codes are in GapExtraction_and_TEanalysis.README 
Detail on dNdS / DoS analysis and codes are in Dos_DnDS_Analysis.README
The code used to calculate Selection coefficient is FitnessVarFreq.R
